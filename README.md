## Statement
水平位置可以选择1到MAX_COLUMN_COUNT个位置选择，垂直Point距离可以自己定制，最后根据所有的Point位置，自动连接所有的点，绘制路径，关于其它一写颜色，宽度，offset偏移，拐角额外距离,拐角方式，文字设置，icon等可以另外配置，具体接入请参考demo
